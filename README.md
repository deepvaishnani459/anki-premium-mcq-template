# 🏆 Premium Anki MCQ Template

Interactive Multiple Choice Question Template for Anki with gamification, streak tracking, milestone rewards, embedded sounds, accessibility support, dark mode, and mobile optimization.

---

## ✨ Features

### 🎯 Accurate Answer Validation
- Exact answer matching
- Reliable option detection
- Supports formatted content

### 🔊 Embedded Audio
- No MP3 files
- No external downloads
- Uses Web Audio API

### 🔥 Streak Tracking
- Persistent learning streaks
- Local storage support
- Motivational feedback

### 🏆 Milestone Rewards
- 3 Correct Answers
- 5 Correct Answers
- 10 Correct Answers
- 20 Correct Answers
- 50 Correct Answers

### ✨ Achievement Popups
- Dynamic encouragement messages
- Celebration animations

### 🌙 Dark Mode
- Native Anki Night Mode support
- Automatic theme adaptation

### 📱 Mobile Optimized
- AnkiDroid compatible
- AnkiMobile compatible
- Responsive design

### ♿ Accessibility
- Keyboard navigation
- Screen reader support
- ARIA labels

### ⚡ Zero Dependencies
- No CDN
- No Frameworks
- No Libraries
- Fully Offline

# 🚀 Installation

## Create Fields

Add these fields to your Anki Note Type:

```text
Question
Option1
Option2
Option3
Option4
CorrectOption
Explanation
```

---

## Front Template

Copy:

```text
templates/front-template.html
```

into:

```text
Cards → Front Template
```

---

## Back Template

Copy:

```text
templates/back-template.html
```

into:

```text
Cards → Back Template
```

---

## Styling

Copy:

```text
templates/styling.css
```

into:

```text
Cards → Styling
```

---

# Example Card

Question:

```text
Which layer handles IP routing?
```

Option1:

```text
Data Link Layer
```

Option2:

```text
Network Layer
```

Option3:

```text
Transport Layer
```

Option4:

```text
Session Layer
```

CorrectOption:

```text
Network Layer
```

Explanation:

```text
The Network Layer handles logical IP addressing and routing.
```
